# OpenClaw Quick Reference Card
## Your AI Employee Cheat Sheet

---

## The 3 Core Files

| File | Purpose | Example Content |
|------|---------|----------------|
| **SOUL.md** | Who your AI is | Values, tone, approach |
| **USER.md** | Who you are | Goals, preferences, constraints |
| **MEMORY.md** | What matters long-term | Projects, relationships, lessons |

---

## The Task Formula

**Weak:** "Help me with X"
**Strong:** "Do X by [time] and deliver to [place]"

**Pattern:**
1. WHAT you want done
2. WHEN you want it done
3. HOW you want it delivered
4. WHERE it should go

---

## The Delegation Levels

| Level | You Do | AI Does |
|-------|--------|---------|
| **1. Supervised** | Everything except execution | Executes with your oversight |
| **2. Scheduled** | Strategic direction | Executes on schedule |
| **3. Autonomous** | Goal-setting only | End-to-end execution + reporting |
| **4. Proactive** | Nothing (review only) | Identifies and solves problems |

---

## 5 Ready Workflows

### 1. Content Creation
```
"Post daily content about [topic] at [time] on [channel]"
```

### 2. Research Assistant
```
"Every [day] at [time], research [topics] and send summary to [place]"
```

### 3. Morning Briefing
```
"Every morning at [time], check [calendar/email/tasks] and send briefing to [place]"
```

### 4. Customer Support
```
"Monitor [channel] during [hours], respond to common issues using [knowledge base], escalate [criteria]"
```

### 5. Income Generation
```
"Research [market], create [product type], set up sales on [platform], promote on [channels]"
```

---

## The Trust Progression

**Week 1:** Supervised execution
**Week 2:** One scheduled task
**Week 3:** Multiple workflows
**Week 4+:** Full delegation

---

## Common Mistakes to Avoid

❌ Using it like ChatGPT (questions, not tasks)
❌ Not setting up memory (repeating yourself)
❌ Being too passive (no cron jobs)
❌ Not pushing further (underestimating capability)
❌ Micromanaging (not building trust)

---

## The Mindset Shift

**OLD:** "I ask AI for help"
**NEW:** "I delegate tasks to AI"

**OLD:** "AI is a tool I use"
**NEW:** "AI is a worker I manage"

**OLD:** "I chat with AI"
**NEW:** "I give AI jobs"

---

## Quick Diagnostic

**If you're always in the loop →** Set up cron jobs
**If you repeat yourself →** Set up MEMORY.md
**If AI only responds when you message →** Set up heartbeats
**If you can't point to concrete value →** Push harder

---

## The Goal

From doing → managing → directing → advising → deciding

Each step reduces your involvement while increasing your impact.
