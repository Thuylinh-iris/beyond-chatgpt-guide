## How to Create Social Images

1. Open each HTML file in Chrome/Edge
2. Press F12 (Dev Tools)
3. Click "Toggle device toolbar" (or Ctrl+Shift+M)
4. Set dimensions to match slide size (1080x1080)
5. Right-click → "Capture screenshot"
6. Save as PNG

**Dimensions:**
- Twitter header: 1500x500 (set custom)
- Carousel slides: 1080x1080 (Instagram square)
- Product mockup: Screenshot as-is
